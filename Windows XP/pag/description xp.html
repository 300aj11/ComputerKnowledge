Description XP

Windows XP comes with a fresh new design and many ways to customize the interface to your liking. There is a new Start Menu that is easy to use and full of new functions; or you can use the classic Start Menu if you prefer. To save space on your task bar you can group running applications according to type. For example, if you have multiple project documents open, you can group them into a pop-up menu instead of all across the task bar. This is a great feature that saves time and task bar space.



Also worth mentioning:

  1. XP can be personalized to allow sharing the same computer with family or associates. Each individual can create their own unique account.
  2. Misbehaving programs will not necessarily cause a computer to crash - each application runs in a separate, protected memory space.
  3. A Transfer wizard enables the movement of files, documents and settings from one computer to another.
  4. The Network Setup Wizard simplifies setting up a local network to share printers, devices, files and an Internet connection.
  5. A Firewall helps guard against intrusion when connected to the Internet.
  6. Windows Movie Maker provides the ability to capture, edit, organize and share video camera movies.
  7. Remote Desktop allows you to use your desktop computer from another computer running Windows 95 or later, giving you access to all of your data and applications.


Windows XP recommended system requirements:

  1. PC with 300 megahertz (MHz) or higher processor clock speed recommended; Intel Pentium/Celeron family, AMD K6/Athlon/Duron family, or compatible processor
  2. 128 megabytes of RAM or higher
  3. 1.5 gigabyte (GB) of available hard disk space
  4. Super VGA (800 × 600) or higher resolution video adapter and monitor
  5. CD-ROM or DVD drive
